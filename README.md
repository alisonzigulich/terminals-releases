# Terminals — Release Feed

Public release distribution + auto-update feed for the [Terminals](https://github.com/alisonzigulich/terminals) macOS app.

- **Appcast (auto-update feed):** [`appcast.xml`](https://alisonzigulich.github.io/terminals-releases/appcast.xml)
- **Latest release:** [Releases page](https://github.com/alisonzigulich/terminals-releases/releases/latest)
- **Source code:** private repo — only release artifacts are public here.

Updates are delivered via [Sparkle 2](https://sparkle-project.org). The app checks this feed automatically every 24h.
